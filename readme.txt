Social media buttons with widget made simple.

== Description ==

This is plugin is a simple social media buttons that appear on all page and post it has also Facebook like box, Google badge, and Twitter embedded feed.

___Whats included:___

**Buttons:**

* Facebook Like Button
* Twitter Button
* Google Plus Button
* Pinterest


**Widget:**

* Social Media Buttons
* Facebook Like Box
* Twitter Feed
* Google Badge

If asking for donations is to much [Hire me](https://www.odesk.com/users/~0193715ab70df6654a) instead.

== Installation ==

1.  Extract 'Social-Media-Button-Widget.zip' to your '/wp-content/plugins/' directory.
2.  In the admin panel under plugins activate 'Social Media Button & Widget'.
3.  You can configure the settings from Social Media Button & Widget' menu under 'Settings Dashboard' _(Make Sure you get your facebook app id)_
4.  Done!
